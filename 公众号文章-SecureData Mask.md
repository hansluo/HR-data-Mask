# HR 必备：一键脱敏员工 Excel，外部协作不再"裸奔"

## 一个真实的痛点

你一定经历过这样的场景——

外部顾问需要一份**部门人员信息表**做组织诊断，但表格里赫然写着每个人的姓名、身份证号、手机号、薪资……

你花了整整一个下午，逐列删改、手工替换，生怕漏掉一个手机号。改完之后，顾问返回了带标注的文件，你又得**一个个把假名字改回真名字**，对照映射表核实半天。

更让人头疼的是——这个过程**每次协作都要重来一遍**。

> 有没有一个工具，能让这件事变成"点一下按钮"的事？

今天，介绍一款我们专为 HR 场景打造的工具 —— **SecureData Mask**。

## SecureData Mask 是什么？

一句话说清楚：

> **一键将 Excel 中的员工敏感信息替换为脱敏值，协作完成后一键还原。纯本地运行，数据不出你的电脑。**

它不是一个通用的数据处理工具，而是专门为 HR 从业者设计的——

- 🎯 **懂 HR 数据**：自动识别姓名、部门、身份证号、手机号、薪资、绩效等级、九宫格等 20+ 种 HR 常见字段
- 🔒 **零泄露风险**：所有处理在你的浏览器本地完成，不联网、不上传、不经过任何服务器
- 🔄 **支持反向还原**：外部团队返回的文件，一键把脱敏值换回真实信息

## 三大核心能力

### 能力一：一键智能脱敏

上传 Excel → 点一个按钮 → 搞定。

系统基于列名自动识别敏感字段，并匹配最合适的脱敏策略：

| 你的数据 | 脱敏后效果 |
|---------|----------|
| 张三 | → 员工1 |
| 技术研发部 | → 部门A |
| 110101199001011234 | → 110\*\*\*\*\*\*\*\*\*\*\*1234 |
| 13812345678 | → 138\*\*\*\*5678 |
| 月薪 25000 | → 随机区间值 |
| Outstanding | → 等级打乱映射 |
| zhang\@company.com | → user1\@company.com |

不满意？你还可以**逐列微调**——切换脱敏方式，或者取消某些列的脱敏。

### 能力二：脱敏记录自动保存

每次脱敏操作，系统都会自动保存一条**脱敏线程**，包含：
- 原始数据
- 脱敏配置
- 脱敏值与原始值的完整映射关系

这意味着：
- ✅ **关掉浏览器数据不丢** —— 下次打开还在
- ✅ 随时回看历史记录
- ✅ 支持导出映射文件（JSON），换电脑也能恢复

### 能力三：反向重塑——最独特的能力

这是 SecureData Mask 真正让人"哇"的功能。

**场景是这样的：**

你把脱敏后的 Excel 发给外部团队 → 他们在上面**新增了评估结果、打分、建议**等列 → 返回给你。

现在你需要：**把脱敏值还原为真实信息，同时保留他们新增的所有数据。**

SecureData Mask 可以帮你一步完成👇

| 脱敏后发出 | 外部团队返回 | 反向重塑后 |
|-----------|-----------|----------|
| 员工1 | 员工1 → 绩效建议：优秀 | **张三** → 绩效建议：优秀 |
| 员工2 | 员工2 → 绩效建议：良好 | **李四** → 绩效建议：良好 |
| 部门A | 部门A | **技术研发部** |

外部团队做的标注**一个字都不会丢**，而你的员工真实信息**一键就回来了**。

## 三个你一定会用到的场景

### 📌 场景一：外部顾问盘点

HR 需要将部门人员信息提供给管理咨询顾问做组织诊断，但不想暴露真实姓名和薪酬数据。

**操作流程：**

3. 反向重塑 → 还原真实姓名 → 导出最终结果

**操作流程：**
1. 上传绩效数据 → 脱敏姓名和部门 → 导出匿名版本
2. 校准完成后收回带调整结果的 Excel
3. 反向重塑 → 还原人员身份 → 落地调整后的绩效结果

### 📌 场景三：培训需求分析

将员工能力评估数据提供给培训供应商做需求分析，但需要保护个人隐私。

**操作流程：**
1. 上传能力数据 → 脱敏姓名、部门、联系方式 → 导出
2. 培训供应商完成分析并添加培训建议列
3. 反向重塑 → 还原真实信息 + 保留培训建议

## 5 步脱敏，3 步还原

**脱敏只需 5 步：**

1️⃣ 打开工具 → 点击「新建脱敏」

2️⃣ 上传 Excel → 拖拽或点击上传

3️⃣ 一键脱敏 → 系统自动识别并处理

4️⃣ 微调（可选） → 预览结果，按需调整

5️⃣ 导出 → 下载脱敏后的 Excel

**还原只需 3 步：**

1️⃣ 选择历史线程 → 找到对应的脱敏记录

2️⃣ 导入回流文件 → 上传外部团队返回的 Excel

3️⃣ 导出结果 → 下载还原后的完整 Excel

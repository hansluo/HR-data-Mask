# SecureData Mask — HR 数据脱敏专家

## 这是什么？

SecureData Mask 是一款专为 HR 从业者打造的 **Excel 数据脱敏工具**。

当你需要将包含员工敏感信息（姓名、身份证号、薪资、手机号等）的 Excel 表格分享给外部协作方（如顾问公司、培训机构、数据分析团队）时，这个工具可以 **一键将敏感数据替换为不可识别的脱敏值**，同时保留数据的分析价值。

> 🔒 **所有数据仅在你的浏览器中本地处理，绝不上传任何服务器。**

---

## 核心功能

### 🚀 一键智能脱敏

上传 Excel 后，系统会 **自动识别 HR 敏感字段**（基于列名），并匹配最合适的脱敏方式。你只需点击一个按钮，即可完成全部脱敏操作。

自动识别的字段类型包括：

| 字段类型 | 示例列名 | 脱敏效果 |
|---------|---------|---------|
| 姓名 | 姓名、员工姓名、中文名 | 张三 → 员工1 |
| 部门/组织 | 部门、组织、团队、中心 | 技术研发部 → 部门A |
| 身份证号 | 身份证、身份证号 | 110101199001011234 → 110\*\*\*\*\*\*\*\*\*\*\*1234 |
| 手机号 | 手机、电话、联系方式 | 13812345678 → 138\*\*\*\*5678 |
| 银行卡号 | 银行卡、账号 | 6222021234567890 → \*\*\*\*\*\*\*\*\*\*\*\*7890 |
| 薪资/金额 | 薪资、工资、年薪、奖金 | 25000 → 随机区间值 |
| 绩效等级 | 绩效、绩效等级、考核结果 | Outstanding → 等级打乱映射 |
| 九宫格/梯队 | 九宫格、人才梯队、继任 | 高潜力 → 等级打乱映射 |
| 邮箱 | 邮箱、email | zhang@company.com → user1@company.com |

**一键完成后，你还可以逐列微调** —— 更换脱敏方式，或者取消某些列的脱敏。

---

### 📂 脱敏线程（Thread）管理

每次脱敏操作会自动生成一条 **脱敏线程记录**，保存了：
- 原始数据
- 脱敏配置
- 脱敏值与原始值的对应关系（映射表）

这意味着：
- ✅ **关闭浏览器后数据不会丢失**（自动保存在本地）
- ✅ 可以随时回看历史脱敏记录
- ✅ 可以导出映射文件（JSON）作为备份
- ✅ 可以在其他电脑上导入映射文件恢复

---

### 🔄 反向重塑（数据回流还原）

这是 SecureData Mask 最独特的能力。

**使用场景**：你把脱敏后的 Excel 发给外部团队，他们在上面新增了评估结果、打分、建议等列后返回给你。现在你需要把脱敏值还原回真实姓名/部门，同时保留他们新增的数据。

**操作步骤**：
1. 打开之前的脱敏线程
2. 导入外部团队返回的 Excel
3. 系统自动将脱敏值还原为真实值
4. 新增的列数据完整保留
5. 导出包含 **真实员工信息 + 外部新增数据** 的完整 Excel

**举例**：

| 脱敏后发出 | 外部返回 | 反向重塑后 |
|-----------|---------|-----------|
| 员工1 | 员工1 → 绩效建议：优秀 | **张三** → 绩效建议：优秀 |
| 员工2 | 员工2 → 绩效建议：良好 | **李四** → 绩效建议：良好 |
| 部门A | 部门A | **技术研发部** |

## 典型使用场景

### 场景一：外部顾问盘点

> HR 需要将部门人员信息提供给外部管理咨询顾问进行组织诊断，但不想暴露真实姓名和薪酬数据。

1. 上传人员信息 Excel → 一键脱敏 → 导出发给顾问
2. 顾问完成诊断后返回带标注的 Excel
3. 反向重塑 → 还原真实姓名，保留顾问标注 → 导出最终结果

### 场景二：绩效校准数据准备

> 需要将绩效数据分享给跨部门评审团进行校准，但希望先隐去具体人员信息。

1. 上传绩效数据 → 脱敏姓名和部门 → 导出匿名版本
2. 校准完成后收回带调整结果的 Excel
3. 反向重塑 → 还原人员身份 → 落地调整后的绩效结果

### 场景三：培训需求分析

> 将员工能力评估数据提供给培训供应商做需求分析，但需要保护个人隐私。

1. 上传能力数据 → 脱敏姓名、部门、联系方式 → 导出
2. 培训供应商完成分析并添加培训建议列
3. 反向重塑 → 还原真实信息 + 保留培训建议

## 快速上手

1. **打开工具** → 点击「新建脱敏」
2. **上传 Excel** → 拖拽或点击上传你的 HR 数据表
3. **一键脱敏** → 点击「一键智能脱敏」，系统自动识别并处理
4. **微调（可选）** → 预览结果，按需调整个别列的脱敏方式
5. **导出** → 点击「脱敏并导出」，下载脱敏后的 Excel

需要反向还原时：
1. **选择历史线程** → 在首页找到对应的脱敏记录
2. **导入回流文件** → 上传外部团队返回的 Excel
3. **导出结果** → 下载还原后的完整 Excel
